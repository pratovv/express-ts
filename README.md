# Как запустить проект:

1. Склонировать репозиторий

git clone https://github.com/pratovv/express-ts

2. Перейти в папку

cd express-ts

3. Установить зависимости

npm install

# Начать работу можно с помощью команды:

1. npm run build 

2. npm run start:prod

либо в режиме разработки 

1. npm run start:dev
